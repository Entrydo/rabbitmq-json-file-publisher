# Import to rabbit tool

## Purpose
Load json file into RabbitMQ consumer 

## Install
1. Create `.env` file (use `.env.example` as template) 
2. Install composer using `composer install`
3. Run the script via `php src/script.php`

## Usage
- `php src/script.php`
